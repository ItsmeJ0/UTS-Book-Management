4. Buat database berikut ini 
**CREATE TABLE books (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    author VARCHAR(255) NOT NULL,
    year INT DEFAULT NULL,
    genre VARCHAR(100) DEFAULT 'Unknown'
);**
